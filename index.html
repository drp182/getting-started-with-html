Getting Started With HTML


The Web

The worldwide web is a collection of HTML documents. There are four main pieces of the Web- Your computers web browser, the internet, HTTP, and servers.Your browser will make a request using HTTP to the internet and the servers will respond with the files that your browser will display. HTTP is that main Protocol of the Web.

HTML

Hypertext Markup Language is is the basis of everything on the web. HMTL is made up of text content (what you see), markup, (what it looks like), references to other documents (images and videos), and links to other pages, such as Udacity.

Computers are Stupid

Computers are sensitive to typos and small mistakes can be exaggerated, such as forgetting to close an end tag, which will leave the remaining words italicized.

Inline Vs Block

HTML elements can either be inline or block. Inline elements just tell when the page should break.
Block elements form an invisible box that can have height and width.
